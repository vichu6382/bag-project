Windows features
------------------
    controlPanel -> UninstallProgram -> Turn Windows features On or Off
    Select(CheckBox) Following
        Windows Subsystem for Linux     (WSL)
        Virtual Machine Platform        (VMP)
        Windows Hypervisor Platform     (WHP)
        Hyper-V 
    Then Press OK Button

Command Prompt ( Open with Administrator Rights)
-----------------
    wsl -v -l   ( Version Check)
        Ouput:
              NAME      STATE           VERSION
            * Ubuntu    Running         2
              Debian    Stopped         2

    wsl --set-default-version 2
        Output:
            For information on key differences with WSL 2 please visit https://aka.ms/wsl2
            The operation completed successfully.

Create New Folder In D: Drive
------------------------------
    mkdir ReactClass
    cd ReactClass

Create React Project
---------------------
    npx create-react-app boopathyapp   (use small case name)
    cd boopathyapp

Error To Create ReactAPP
-------------------------
	"npm error enoent ENOENT: no such file or directory, lstat 'C:\Users\Home\AppData\Roaming\npm'  " 
	Manually creating a folder named 'npm' in the displayed path To fixed the problem.



Start Node (npm) Inside of boopathyapp Folder
----------------------------------------------
    npm Start
    (D:\ReactClass\boopathyapp> npm start)
    Your Browser Will Open with index.html page available in Public Director of React Package:
        http://localhost:3000
        http://127.0.0.1:3000
